```
1. 面向对象编程
	分析对象之前的关系，主要有以下几种：
	1>has-a composition 具体可以分为两种aggrevation(聚合关系) 和composition(组合关系）
		组合关系例如树和树叶，一片树叶智能属于一棵树，树叶随着树的产生，也随着树的死亡而死亡，一片叶子是树的一部分。树叶的
		的死亡，树并不会死亡。
		聚合关系例如车和车轮，一个车必须有车轮，一个车轮可以安在不同的车辆上，车报废了该车轮不一定报废了；车轮爆了车不一
		定报废。
	2>use-a assotation 联系关系
		战士和枪，独立率彼此存在，一个战士可以拥有多支枪，一只抢可以由许多战士使用；
	3>depends-on dependency 依赖关系
		人和空气，人依赖于空气（单向关系），被依赖的人产生基于依赖者的需要，一旦被依赖者被破坏，依赖者将会丧失某个功能；
	4> member container 包含关系
		array 和item的关系
	5> is-a inherit 继承关系
		父子关系，继承传代；
```
